# IMDB Top 1000 Movies Analysis

## About this project
Exploratory Data Analysis on the IMDB Top 1000 Movies dataset
to understand patterns across genres, directors, actors,
ratings, and box office performance.

## Questions explored
- Which directors have the highest total gross collection?
- Which genres appear most frequently in top rated films?
- Is there a relationship between votes and gross collection?
- How have movie runtimes and ratings changed across decades?
- Which actors appear most in high grossing films by genre?

## Key findings
- Most top rated movies run between 100 and 140 minutes
- Drama is the most frequently occurring genre after genre splitting
- Votes and gross collection show a positive relationship
  with notable exceptions on both ends
- Average ratings have remained stable across decades

## Tools used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
IMDB Top 1000 Movies dataset from Kaggle
